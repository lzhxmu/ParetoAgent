# agent-efficiency-booster 是什么

`agent-efficiency-booster` 是一个给 Claude Code / Codex 使用的效率优化 skill，目标是：

- 降低 Agent 在真实任务中的 token/API 成本
- 提高任务完成速度
- 在可控范围内保持质量稳定

## 这个库做什么

它提供一套可执行的效率策略：

1. **预算优先**：先设定成本、时延、质量约束，再执行任务。
2. **ROI 分流**：把子任务按“耗时/花费/复杂度”排序，优先优化最值钱环节。
3. **模型与工具路由**：重任务用强模型，轻任务优先脚本/低成本模型。
4. **上下文压缩**：减少无关上下文，避免长上下文导致成本与时延上升。
5. **并行与止损**：只并行独立任务；当边际收益下降时提前停止。
6. **质量守护**：至少执行语法/测试/变更范围检查，避免“省钱但返工”。

## 仓库当前内容

- `skills/agent-efficiency-booster/SKILL.md`：skill 使用说明与标准工作流。
- `skills/agent-efficiency-booster/scripts/roi_triage.py`：ROI 排序脚本（输入任务 JSON，输出优化优先级）。

## 预期效果（短期）

- 成本下降：约 20%~40%
- 时延下降：约 15%~35%
- 质量变化：通常可控制在 5% 以内（取决于任务与校验强度）
